Glossary

HTML
HTML stands for HyperText Markup Language and is not a programming language. It consists of elements that 'wrap' the content of a web document. The start and end of elements are indicated using tags
HTML attributes
An HTML element may also have attributes, which contain information about the element beyond the content and can be used to identify an element for styling and in scripts. HTML attributes consist of a name followed by an equal sign and then a value enclosed in quotation marks.
CSS
CSS stands for Cascading Style Sheets, this is used to specify the layout of structural HTML elements, appearance of HTML elements, and can even do some animation. CSS is not a programming language, it simply describes the visual appearance of specific elements.
Wireframe
A wireframe is a visual guide that represents the skeletal framework of a website otherwise known as screen blueprint. The term wireframe is taken from other fields that use a skeletal framework to represent 3 dimensional shape and volume.
GitHub
GitHub is an internet hosting service for software development and version control using Git. It provides the distributed version control using Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.
